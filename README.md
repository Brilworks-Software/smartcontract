# Smart Contract

Ethereum smart contract development with Solidity.

## Tech Stack

- **Language:** Solidity, JavaScript
- **Tools:** Node.js for compilation and deployment

## Files

- `contracts/` — Solidity smart contracts
- `compile.js` — Contract compilation script
- `deploy.js` — Deployment script
- `test/` — Contract tests

## Getting Started

```bash
npm install
node compile.js
node deploy.js
```
